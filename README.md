## Usage
1. Create a public repo (e.g., `microapps-demo`).
2. Copy this folder content to the repo root.
3. Enable GitHub Pages: Settings → Pages → Source: `main` → `/root`.
4. Adjust domain references if needed.

## Canonical & Duplicates
Если GitHub Pages дублирует контент сайта, используйте `<link rel="canonical" href="https://microapps.by/...">` на соответствующих страницах. В этом демо теги уже добавлены в `index.html` и `demo/service.html`.
